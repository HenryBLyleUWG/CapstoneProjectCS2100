# STEM Hub Website for CapstoneProjectCS21000

STEM Hub is an educational website designed to help students explore Science, Technology, Engineering, and Mathematics through projects, learning resources, activities, and community engagement.

## Features
- Responsive design
- Image gallery
- Interactive details widget
- JavaScript alerts and validation
- E-commerce products page
- GitHub Pages deployment

## How to Use
1. Open the website using the deployed GitHub Pages link.
2. Navigate through the menu to explore different sections.
3. Browse projects, shop products, and explore learning resources.

## Technologies Used
- HTML5
- CSS3
- JavaScript
